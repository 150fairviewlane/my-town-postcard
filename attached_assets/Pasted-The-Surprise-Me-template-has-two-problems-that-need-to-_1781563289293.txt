The "Surprise Me" template has two problems that need to be fixed together in this prompt:

PROBLEM 1 — Sequencing bug causing bad output:
The templateKey resolution (from "surprise-me" to a real template name like "sage-organic") happens after refLines and outputRequirements are already built. So Surprise Me always uses the generic fallback prompt regardless of which template was randomly selected. The template reference image and the prompt instructions are mismatched, causing Grok to produce incoherent output.

PROBLEM 2 — No visual distinction from the base template:
Even after fixing the sequencing bug, Surprise Me would produce identical output to just selecting that template directly. We want Surprise Me ads to look subtly but noticeably different from the base template while still being clean and professional.

Fix the sequencing bug AND apply the three visual variation instructions described below, all in the same update.

VARIATION 1 — Font swap:
In the TYPOGRAPHIC RULES section of the outputRequirements, find the headline font instruction. It will say something like "bold condensed all-caps slab serif" for most templates, or "bold condensed sans-serif" for others.
- If the base template uses "slab serif" in the headline rule, replace that phrase with "bold condensed all-caps geometric sans-serif"
- If the base template uses "sans-serif" in the headline rule, replace that phrase with "bold condensed all-caps slab serif"
This swap applies only to the headline font description in the typographic rules. Do not change any other font instructions.

VARIATION 2 — Accent color on the coupon:
In the outputRequirements, find the coupon or special offer zone instruction. After the coupon zone description, add this sentence:
"Use the industry accent color (the brighter or warmer of the two brand colors) as the dominant color for the coupon box background or border, rather than the primary color."
If there is no coupon zone because the business has no offer, skip this variation silently.

VARIATION 3 — Coupon style swap:
In the same coupon zone instruction, find the style description and replace it based on what the base template normally uses:
- If the base template uses a dashed border coupon style, change the description to: "solid filled coupon block with rounded corners, the offer text in large bold white letters on a filled background"
- If the base template uses a ticket-stub or torn-edge coupon style, change the description to: "clean rectangular coupon with a bold solid border and a subtle diagonal stripe pattern on the left edge"
If there is no coupon zone, skip this variation silently.
