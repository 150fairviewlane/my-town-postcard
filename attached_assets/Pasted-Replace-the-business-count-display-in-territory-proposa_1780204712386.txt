THE CHANGE
════════════════════════════════════════════════

REMOVE from the proposal card and mailing area list:
  - "Ad-Ready Businesses: X,XXX"
  - "X businesses" under each mailing area
  - Any reference to a specific business count number

REPLACE with a single market tier badge on the proposal card:

  Determine tier from the hub cities' combined population:

  totalHubPopulation = sum of Gazetteer POPULATION for all 4 hubs

  if totalHubPopulation >= 500,000:
    tier = 'Metro Market'
    color = burgundy (#7C1C2E)
    description = 'High-density market with strong advertiser demand'

  else if totalHubPopulation >= 150,000:
    tier = 'Strong Market'
    color = green (#1a5c3a)
    description = 'Established market with solid advertiser base'

  else if totalHubPopulation >= 50,000:
    tier = 'Good Market'
    color = #2563eb (blue)
    description = 'Growing market with good advertiser opportunities'

  else:
    tier = 'Developing Market'
    color = #d97706 (amber)
    description = 'Smaller market — ideal for a motivated dealer'
