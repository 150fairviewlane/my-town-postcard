════════════════════════════════════════════════
HOW THIS TOOL WORKS
════════════════════════════════════════════════

This is a different architecture from the previous ad generators.
It does NOT call the OpenAI API directly. Instead:

1. The user fills in their business details on the left side
1. The system builds a detailed prompt with strict text accuracy rules
1. The user clicks Launch AI Console — this opens claude.ai or
   chatgpt.com in a popup window sized to sit alongside the app
1. The prompt is automatically copied to the clipboard
1. The user pastes the prompt into the AI console and hits send
1. The AI generates a professional ad image in the console
1. The user right-clicks the result and saves it, then drops it
   into the result panel on the right side of the page
1. The user clicks Use This Ad to save it to the database

The popup window opens at screen.width minus popup width minus 20px
from the right edge, so it sits beside the main app window.

No API keys are needed for this feature. No OpenAI costs are incurred
by the platform. The user’s own Claude or ChatGPT account handles the
image generation.
