WHAT CHANGED IN THIS VERSION
════════════════════════════════════════════════

1. FOOD PHOTO is now a prominent foreground element
   Previously the photo was a faded background blend using mix-blend-mode.
   It is now a real img element occupying the right 73% of the card at
   z-index 2 (above the template, below all text). Gradient edges blend
   it naturally into the parchment template.
1. BUSINESS NAME typography now matches the original reference ad style
   Line 1 (company name) is in Bebas Neue bold, tilted -1.5 degrees
   Line 2 (script word) is in Pacifico cursive, larger than Line 1,
   tilted -3 degrees, with a negative margin-top so it overlaps the
   bottom of Line 1 — exactly like the original reference ad design
   Decorative tick marks added to the top right
1. ALL OVERLAY POSITIONS are now pixel-scanned from the actual template
   Checkmark circle centers measured at y=727, 832, 932, 1037 pixels
   Menu text left edge at 10.5% of container width
   Coupon box boundaries measured from actual dashed border pixel positions
   Footer dark bar measured at y=1132 pixels (82.57% from top)
1. INDUSTRY SWITCHING now always updates all fields
   Switching industry always repopulates tagline, all 4 menu items,
   offer amount, offer description, and fine print — not just on
   first selection
