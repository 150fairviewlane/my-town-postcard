The Smart Generator hides all template selection from the user. Instead:
- The system automatically picks the best template for the selected industry
- The user picks from 6 pre-defined color pairs that inform Grok's color treatment
- The user fills in their business info, uploads optional photo and logo
- They click "Generate My Ad" up to 6 times, each time getting a new ad using the next-ranked template
- All generated ads accumulate as thumbnails in a gallery row
- After 6 generations, the Generate button is disabled
- The user clicks any thumbnail to enlarge it (lightbox) and select it
- Selecting an ad goes straight to reservation checkout

When the user clicks Generate My Ad:
1. Look up the ranked template list for the selected industry
2. Remove any templates already used on this campaign side (query the same used-templates endpoint the classic generator uses: GET /api/campaigns/:campaignId/used-templates?spotId=:spotId)
3. Remove any templates already used in THIS generation session (tracked in frontend state as an array of used template keys, reset on page load)
4. Pick the first remaining template from the ranked list
5. If all 6 ranked templates are exhausted (extremely rare), loop back to rank 1
6. Pass the selected template key to the backend exactly as the classic generator does — the backend prompt-building logic is unchanged

Track which template was used for each generated ad and display it as a small label under each thumbnail (e.g. "Neighborhood Pro") so the user can see variety.
